# mc-manager
MC Manager est une solution complète et open-source conçue pour simplifier radicalement l'administration de serveurs Minecraft.

Fini de jongler avec des scripts .bat ou des lignes de commande obscures. MC Manager offre une interface intuitive pour déployer, configurer et surveiller vos instances en temps réel. Que vous soyez un administrateur gérant un réseau de serveurs ou un joueur souhaitant héberger une partie privée, cet outil centralise tout : gestion de la RAM, installation de plugins/mods, et sauvegardes automatiques.

Pourquoi l'utiliser ?

Simplicité : Lancez un serveur en moins de 3 clics.

Flexibilité : Compatible avec Vanilla, Spigot, Forge, Fabric et Paper.

Sécurité : Système de backup automatisé pour ne jamais perdre votre monde.
